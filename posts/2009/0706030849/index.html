Senna 使って、夜の動画検索サイト「sugokuii beta」を作ってみた

Posted at — Jul 6, 2009

ムフフな夜の動画（今のところ YourFileHost のみ対応）をキーワードで検索できるサイト 「sugokuii beta」というものを作ってみました。以前作った YourAVHost とどこが違うのか と思われるかもしれませんが、YourAVHost は新着動画の情報を伝えるブログ的なものだったのに対して、 sugokuii の場合は特化した検索エンジンのような形になっております。例によって18禁。

sugokuii beta - 夜の動画検索エンジン

最近主要検索エンジン各社がどうも動画検索に力を入れているらしくて、 Bing や Yahoo! JAPAN のものなんて、結果がでたらその場で再生なんかできちゃったりして、 結構楽しいです。そこで、当然ながら好きな AV 女優の名前なんかいれ、 ムフフな動画をみつけて喜んだりするわけですが、大きな問題点がひとつあります。それは

「肝心の（夜のお供の）YourFileHost の動画が出てこないじゃないか！」

ということです。 そこで YourFileHost の動画もでてくる、いやむしろそれしか出てこない「夜の」 検索エンジンを自分で作ってしまえというのが今回のモチベーションになりました。

猛烈な勢いで検索対象のファイルを追加していっているのですが、 現在のところ、「2万件強」の動画の中から好きな言葉で検索して、 サムネイルつきの結果を見ることができます。 また、インデックス数の関係からまだ精度があまりよくないのですが、 ファイルごとに関連する動画の一覧表示をする機能もあり、 シリーズものなども一気に閲覧できるように心がけております。

例えば 「柚木ティナ ゴスロリ」というキーワードを Google のビデオ検索にかけても 期待するものは出てきませんが、sugokuii ではお目当てのビデオもひっかかりますね。 ちなみにこの「柚木ティナ ゴスロリ」的な動画、自分でオリジナルなものを持っているのですが、 「すごくいい」のでよくお世話になっておりまして特に...（ry

sugokuii beta - 柚木ティナ ゴスロリの動画検索結果

では、仕組みについて簡単に解説します。 タイトルにあるように全文検索には Senna を使っています。 MySQL に既にバインディングされている「Tritonn」を使えばインストールも簡単にできて、 クエリーもわかりやすくかつ、パフォーマンスも今のところ気にならないのですごくいい感じです。

さて、肝はこの検索をするための文章をどうやって持ってくるか、 そして動画の情報とどう結びつけるかになります。 前々から YourFileHost の URL を投げると AV女優名が返ってくるというシステムを作りたかったのですが、 よく考えたらそれを逆に応用して結果を残しておけば AV女優名やキーワードを投げると YourFileHost のURL を取得する 完全な検索エンジンができるなと思いました。

どう実装するか。実は私、こういうものをいろいろと作っている関係柄、 もちろん延べで重複を含みますが約 2000 万件の YourFileHost の個別動画ファイル名、つまり ID を所有しておりましてw （重複を除いた実際のファイル数はインデックス張ってカウントするのに時間かかりそうなんでやってません＞＜）、 それを利用します。 材料が揃うと次に、このリストの中から一つずつファイル名を取り出してきて、 ファイル名もしくは、URL の被リンクのクエリーを、一般検索エンジンの API に投げます。 するとその動画にリンクしたり関連するサイトのタイトル及びサマリーが返ってくるわけで、 それを実のところその動画に関するメタデータとして扱おうという仕組みです。 返ってきたタイトル及びサマリーを Senna の入った DB に入れて、最初にキーとして渡した動画の ID と結びつければ、 全文検索のためのテーブルの準備ができます。 この作業をひたすらリスト一覧から繰り返していけば、

そこそこ多くの URL を網羅したデータが出来上がるのです。 ここまでくれば、Senna、Tritonn のチュートリアルにあるような KWIC（キーワードインコンテキスト）出力を含む、 検索クエリーを投げるように DBIC の resultset の扱い方を工夫すれば OK、でした。

なんか気になる女優さんができた、もしくはなかなか私の趣味にあうムフフな動画ないんだよね、 という方。まだインデックス作り中で十分な検索結果が得られないかもしれませんが、 とりあえず気になったら sugokuii にキーワードを入れてみてください。 もしかして、お目当ての動画がサクッと見つかるかもしれませんね。

というわけで、夜の動画検索サイト「sugokuii、まだ beta 版」の紹介でした。是非とも、Enjoy!
