Fixed bug on WebService::Simple::Google::Chart 0.04

Posted at — May 23, 2009

またまた、コメントより。 今回は、WebService::Simple::Google::Chart の紹介記事にて、頂きました。

color99 さん:

コードサンプル通りで動作確認しようとしたら、
Odd number of elements in hash assignment at xxxxx
というエラーメッセージが出ます。

う、ごめんなさい。それバグです＞＜ 修正して、CodeRepos にコミット済み。また、ヴァージョン 0.05 として、 CPAN にもアップしておきました。そのうち更新されると思います。 ということで以下のサンプルが動きます。 

 use WebService::Simple::Google::Chart;

my $chart = WebService::Simple::Google::Chart->new;
my $url = $chart->get_url(
    {
        chs => "250x100",
        cht => "p3",
   },
    { foo => 200, bar => 130, hoge => 70 }
);

print $url . "\n";
