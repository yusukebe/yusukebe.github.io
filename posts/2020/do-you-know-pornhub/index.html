Posted at — Jan 21, 2020

Pornhubをご存知だろうか。 答えは当然Yesだろう。 ところがその本当の実態について我々は理解しているだろうか。 WikipediaのPornhubページには興味深い事柄が書いてある。

  • YouTubeに次ぐ世界第2位の動画共有サイト
  • 360度動画を提供しており、PlayStation VRで視聴できる
  • 上下運動で発電できるリストバンドを発明した
  • 宇宙空間でポルノを撮影すると発表した
  • VPNHubというVPNサービスを開始した

これらのいくつかを恥ずかしながら私は知らなかった。 この恥ずかしさがモチベーションとなり、私はPornhubのポルノ以外のことを調べた。 出てくるものはアダルトコンテンツばかりだったが、 以下の3つのページに出会うことができた。

  • Pornhub Insights :: Digging deep into the data
  • Interview with a Pornhub Web Developer
  • Pornhub case study: 5 marketing steps that made it #1

Pornhubの実態

Pornhub Insightsでは節目ごと、国別に統計を事細かくまとめている。The 2019 Year in Review – Pornhub Insightsでは昨年、2019年について書かれている。 興味深い点をピックアップした。これでPornhubがいかにすごいのか、どのように利用されているかが分かるだろう。

Pornhubを支える技術

Interview with a Pornhub Web Developerは 2019年10月の記事である。PornhubのWeb開発者（おそらくフロントエンド）にインタビューをしている。 Pornhubで使われている技術、アダルトサイトで働くということはどういうことかを聞いている。 関心のある部分をピックアップした。といってもだいたい全てである。

サードパーティの広告スクリプトが動いてるが開発の時にどうしているか

  • プレイヤーを2つに区切って開発している
    • コア機能とイベント発火
    • 広告無しの状態で開発される
  • 本番サイトでサードパーティスクリプトと広告を統合している
    • 開発の早い段階で問題を発見したい
    • なので、広告会社にイベントトリガーの手動設定ができるようにお願いしている

動画、GIFの広告、ライブカメラのプレビュー、サムネイル…。それらをどうやって計測して、どうやってパフォーマンスを維持しているか

  • プレイヤーがプレイバック情報をバックエンドに送っている
  • サードパーティのRUM（リアルタイムユーザーモニタリング）システムが基本
  • AWSにWebPageTestのスクリプトが置いてある
    • それを見て何が起こっているのか見ている
    • 場所別、プロバイダー別のwaterfallを解釈できる

ビデオプレイヤーについて。動画再生の前に広告がでる、ハイライト（盛り上がってるところ）が分かる、再生速度の変更ができるけど…

  • ビデオプレイヤー専門のチームがいる
  • ブラウザのパフォマンス計測ツール、Web ページのテスト、メトリクスなどを使う
  • QAがいる

ビデオチーム、フロントエンドチーム は何人くらいいるの？

  • おれが思うに平均的なほうだと思う

フロントエンドについて

  • CSS直書きからLESSを使いだした
  • メディアクエリーで解像度別に対応
  • jQueryとjQuery UIが遅い。vanillaJSを使いだした。OOPできるし興味深い
  • IntersectionObserver APIが大好き。画像のロードに便利
  • Picture-in-Picture APIを使いだした

使っているWeb技術は？

  • Beacon
  • Fetch
  • WebRTC
  • Service Worker

WebVRについてどう思ってる？

  • webXRについての調査をしている
  • VRコンテンツのクリーター、ユーザーが必要
  • PornhubはVR、コンピュータービジョン、バーチャルパフォーマーをサポートした最初のプラットフォーム
  • これからも新しい技術を探求していく

デスクトップとモバイルとかデバイスの違いについて

  • 例えば、iOSでのビデオ再生はフルスクリーン中をカスタム プレイヤーで操作できない
    • ネイティブのQuickTimeプレイヤーで動作するから
    • Androidはフルスクリーンで色々できる
  • HLSによるストリーミング
    • IEとEdgeはいけてない

IE対応について

  • IE11以前のサポートは切った
  • Flashプレイヤーもやめた
  • Chrome、Firefox、Safariに焦点を当てている

サーバーとフロントエンドで使ってる技術を教えて

  • バックエンド

    • Nginx
    • PHP
    • MySQL
    • Memcached
    • 他にも、Vernish、ElasticSearch、Node.js、Go、Verticaなんかを使ってる

  • フロント

    • jQueryの代わりにvanilla Javascript
    • 他のフレームワークも試している。Vue.jsとか

アダルトサイトで働くってどう？

  • 俺は気にしない。とても魅力的なことだ
  • たくさんの人が利用するのがモチベーションになっている
  • ポルノは絶対死なない。これは声を大にして言いたい！！

フロントエンドエンジニアに向けて言いたいことは？

  • 大きなプロダクトのユーザーエクスペリエンスを作ることはとてもエキサイティング
  • 俺たちは技術の最先端を走ってる。これからも継続していくぜ
