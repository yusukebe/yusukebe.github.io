ワディットのホームページを新しくしました

Posted at — Feb 20, 2008

えー、一応俺が代表取締役を務め、親父と共にやっている株式会社ワディットのホームページを新しくしました。 今までは、ただ 1枚 の HTML だったんですが、多少ボリュームアップさせて、事業内容が詳しく分かるようにしました。 一年位前に「おし、作るぞ」といって MovableType のライセンスだけとってほったらかしていたものにようやく手をつけた形です。お寒いトップの写真は庭で母に撮ってもらいました。

業務プロセス改革･改善実践のためのBPMセミナー: セミナー・展示会 | 日商エレクトロニクス より

親父が昔からやりたがっていて、熱心に語り続けていることが形になりつつあって息子としても嬉しいところです。プレゼン頑張ってください。ちなみに、明日はしないらしいのですがデモの環境作りを手伝っていたりして、よく父は、Thinkpad X60 の中に

  • Windows 2003 Server + アプリケーションサーバー on VMWare
  • Linux + Apache + CGI on VMWare
  • Plone ( Zope によるアプリケーションサーバー )

という3つのサーバーを同時に起動してデモしてたりします。

ホームページを新しくしたといっても、割と適当で、まだ内容がつかみきれないかもしれませんが、 お仕事に限らず何かご相談等ございましたら、ご連絡ください。 よろしく！
