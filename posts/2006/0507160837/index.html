ゆーすけべー日記イメージ検索で遊んでみた

Posted at — May 8, 2006

さっき作ったゆーすけべー日記イメージ検索、 いろいろなキーワードで検索して遊んでみた。やっぱり写真がどばって出てくると昔のこととかすぐに思いだされて面白い。 個人的にはこんなキーワードが面白かった。

  • Regunary - yusukebe.com Image Search
    フットサルチームの名前だからフットサルしている画像がたくさん出てくる。
  • お笑い！企画放送局 - yusukebe.com Image Search
    おわきかの過去の作品が思い出される。
  • 奥出研 - yusukebe.com Image Search
    コミュニティの名前で検索するのもあり
  • ORF - yusukebe.com Image Search
    ORFというのはSFCの研究発表会みたいなもの。その準備期間のどたばたぶりが見れて面白い。
  • アンフィ - yusukebe.com Image Search
    サッカーサークル。
  • 湘南 - yusukebe.com Image Search
    湘南高校関係から湘南地区のことまで出てくる。わりとその辺の写真が多い。
  • ラーメン - yusukebe.com Image Search
    ラーメンの写真がいくつか出てくる。が、他にも変なのがでてきてうける。
  • ナタミ - yusukebe.com Image Search
    夏休みに10日間くらい鎌団子の実家で友達を招きまくるという伝説のイベント「ナタミ」。その当時の楽しい思い出がよみがえってくる。
  • 夏 - yusukebe.com Image Search
    「夏」で検索してるのに雪の写真が出てくるのがうける。
  • うける - yusukebe.com Image Search
    俺が「うけた」記事からの写真。
  • 楽しい - yusukebe.com Image Search
    俺が「楽しんだ」記事からの写真。こういう感情キーワード入れるのもいいね。
