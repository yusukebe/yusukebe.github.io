WB Editorから書き込みテスト

Posted at — Mar 8, 2004

WB Editor
クライアントのソフトからMovabletypeのエントリーをポストできるという代物。
今そのソフトから書き込んでおります。はたして書き込まれるのやら。。。

うへ！書き込まれてる！テキストの色変えたりするの簡単！

画像とかファイルのアップロードはどうなんだろうと試行錯誤。いじってみたところ画像を扱う系のボタンとかはなさそう。
とWB Editorのウェブをみてみたらロードマップのところにこう書いてあった。

WB Editor 1.0 - Stand alone editor

Features:

  • Category list
  • New Post
  • Post retrieve, edit and delete
  • Copy IE proxy settings
  • Threaded server connection
  • Editor Options : UI culture, editor/xml charset
  • Trackback ping
  • Automatic self-upgrade
  • Graphic file upload
  • Plugins

1.0ではどうやら画像のアップロードに対応するらしい。こりゃ期待じゃ。

追記

preタグとか使うとうまく改行されない。どうやら改行コードを送ってくれないみたい。
あと画像のアップロードに対応するっていってもこれXML-RPCで通信するんだろ。画像ファイルってXML-RPCで送れたりするのだろうか。
