Twitter社へのお願い「Streaming API links の解放を」

Posted at — Jun 18, 2010

TwibというTwitterの主に日本語ツイートではてなブックマークのような、 人気ウェブサイトを閲覧できるサービスを運営しているのだけど、どうも今のところのやり方では対象となるツイートの取りこぼしが多い。ようはSearch APIとStreaming APIのtrackメソッドを利用して「http」で検索しているわけだけど、 やはりインターバルを短くしたりしても限界があるみたいだ。 そこで先ほど非公開である Streaming API の links method の解放をお願いするためにTwitter社にメールを出してみた。 これをするとStreaming APIで基本的に取りこぼし無しでウェブサイトへリンクを張ったツイーツを取得できる。

実際、今のTwibには取りこぼしがあるのがどうもひっかかっていて、 いろいろと改善したい点や機能追加したい点があるのに、 なかなか手をつけていないで無駄にスパムサイトみたいになっていてすごく気持ちが悪いので、 複雑な心境だった。取りこぼしが無ければ、いい方向に進めると思い、調べてる最中に links の API の件を知って メールを出した。 以下がメールの全文である。このAPIを解放してもらえば、いいサイトにしていきたいので、 もし賛同して頂ければ、以下のツイートをRTしてもらったりすると嬉しいです。
